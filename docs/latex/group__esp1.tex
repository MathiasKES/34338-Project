\doxysubsection{Detailed Description}
This firmware runs on an ESP-\/based Arduino-\/compatible board and implements the first stage of a distributed door access control system.

Hardware components:\+
\begin{DoxyItemize}
\item MFRC522 RFID reader (RC522)
\item I2C LCD display
\item PIR motion sensor
\end{DoxyItemize}

Functional responsibilities:\+
\begin{DoxyItemize}
\item Detects motion to manage LCD backlight and user interaction
\item Reads RFID tags and initiates authentication
\item Displays system and authentication status on the LCD
\item Communicates authentication requests and results via MQTT
\end{DoxyItemize}

RFID authentication must succeed before the keypad stage is enabled.
