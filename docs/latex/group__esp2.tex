\doxysubsection{Detailed Description}
This firmware runs on an ESP-\/based Arduino-\/compatible board and implements the second stage of the access control system.

Hardware components:\+
\begin{DoxyItemize}
\item 4x4 matrix keypad using the Keypad library
\end{DoxyItemize}

Functional responsibilities:\+
\begin{DoxyItemize}
\item Accepts PIN input from the user
\item Provides keypress feedback (beeps)
\item Publishes PIN entry progress and submissions via MQTT
\end{DoxyItemize}

The keypad is only active after successful RFID authentication, which is signaled asynchronously via MQTT.
