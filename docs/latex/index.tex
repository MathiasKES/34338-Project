This project implements a distributed door access control system using multiple ESP8266 nodes communicating over MQTT.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{System Overview}{System Overview}}\label{index_autotoc_md1}
The system consists of three ESP8266 nodes:\+


\begin{DoxyItemize}
\item {\bfseries{ESP1 – RFID \& Motion}} Handles RFID authentication, LCD feedback, and motion sensing.
\item {\bfseries{ESP2 – Keypad}} Handles PIN entry and reports keypad interaction.
\item {\bfseries{ESP3 – Door Actuator}} Controls the door lock servo, LEDs, and buzzer.
\end{DoxyItemize}

A shared {\ttfamily Wi\+Fi\+Mqtt\+Client} library provides a unified abstraction for Wi\+Fi and MQTT communication.
